POO - PHP

La POO (Programmation Orientée Objet), est une façon de programmer, cela a pour interet d'améliorer l’organisation du code et la sécurité des données.

Twig/Monolog

Monolog permet de créer des logs grâce auquel on peut garder une trace de certaines actions faites par les utilisateurs ou le code. Twig est un moteur de templates pour le langage de programmation PHP, utilisé par défaut par le framework Symfony.

Le HTML

Html5 est le langage de balisage le plus utilisé pour la création de site web, cette documentations vous donne les principales balises à connaitre, avec les nouvelles balise intégré depuis html5.

Vue.Js

Vue.js, est un framework JavaScript open-source utilisé pour construire des interfaces utilisateur et des applications web monopages. L'outil est très puissant et permet de simplifier, et optimiser énormément de chose. Cette documentation vous explique les bases de Vue.js, si vous souhaitez avoir un aperçu de ce que l'on peut faire voici un exemple que j'ai réalisé : Je test Vue.js

kléopatra

Grâce à Kléopatra, vous pouvez crée une clé de chiffrement. Elle peut être utilisée pour chiffrer des mail, afin qu’il n’y ais personne que vous et votre destinataire qui puissent voir votre mail.

Docker

Docker est un logiciel libre permettant de lancer des applications dans des conteneurs logiciels.
